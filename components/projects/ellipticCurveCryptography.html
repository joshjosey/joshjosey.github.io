Elliptic Curve Cryptography

This project implements an elliptic curve cryptography (ECC) system in MATLAB. The program allows two users to securely exchange encrypted messages using ECC-based key exchange. The encryption process involves generating a shared elliptic curve, selecting public and private keys, and performing point multiplication operations to encode and decode messages. The project demonstrates key cryptographic principles, including modular arithmetic, group operations on elliptic curves, and secure message encryption and decryption. One of the key advantages of ECC over traditional RSA encryption is its superior security-to-key-size ratio. ECC provides significantly stronger security with much smaller key sizes compared to RSA. For instance, achieving 80-bit security requires a 1024-bit RSA key, whereas ECC achieves the same security level with just a 160-bit key.
